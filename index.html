Introduction

This page shows a basic concept for a floating action button which uses components from the Angular Material Design project.

Example

Below is a basic prototype of the FAB with a speed-dial presentation. It currently supports, md-direction="down" (the default) and md-direction="up" and properly handles keyboard navigation in both cases (via tabs; support for arrow keys is possible but not yet implemented).

Check out the source code of this page (or the project on GitHub) to see how the code is used.

Next Steps
